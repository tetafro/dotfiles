# Dotfiles

A collection of dotfiles and scripts I use for customizing my OS.

## Install

```sh
cp .bash_aliases \
    .bashrc \
    .gitconfig \
    .gitignore \
    .golnagci.yml \
    .inputrc \
    .profile \
    .tmux.conf \
    $HOME
cp vlc/* $HOME/.config/vlc/
cp vscode/* $HOME/.config/Code/User/
cp flake8 $HOME/.config
```
